# Tabbed layout

Tabbed layout gives every window the full container area and displays a tab strip for switching among the group. It is useful when several related windows should occupy the same region without being visible at once.

## Use tabbed layout

| Action                     | Default shortcut                               |
| -------------------------- | ---------------------------------------------- |
| Toggle tabbed layout       | <kbd>Shift</kbd>+<kbd>Super</kbd>+<kbd>T</kbd> |
| Show or hide the tab strip | <kbd>Ctrl</kbd>+<kbd>Alt</kbd>+<kbd>Y</kbd>    |

Directional focus shortcuts activate other windows in the group. Moving or dragging windows can add them to or remove them from the tabbed container.

## Automatic exit

**Auto exit tabbed tiling** is enabled by default. When a tabbed container is reduced to one window, Anvil returns it to a normal split layout automatically. Disable that option under **Anvil Preferences → Tiling → Behavior** if you want the single remaining tab to keep its tabbed container.

## Drag and drop

Set **Drag-and-drop behavior** to **Tabbed** to make a center drop create or join a tab group. This is the default center-drop action.

Tabbed layout support and tab decoration can be controlled independently in preferences.
