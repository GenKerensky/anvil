# Stacked layout

Stacked layout gives every window the full container area while keeping a title-bar strip for the windows in the stack. The focused window is raised above the others.

## Use stacked layout

1. Focus a window inside the container you want to change.
2. Press <kbd>Shift</kbd>+<kbd>Super</kbd>+<kbd>S</kbd>.
3. Use normal focus shortcuts to move through the stack.

Press the shortcut again to return to a split layout. Leaving stacked layout redistributes the child sizes so the container can tile normally again.

## Drag and drop

When **Drag-and-drop behavior** is set to **Stacked**, dropping a window in the center of another tile creates or joins a stack. Edge drops detach a window from the stack and place it into a split.

Enable or disable stacked layout support from **Anvil Preferences → Tiling → Behavior → Stacked tiling**. The feature is enabled by default and currently marked experimental in preferences.
