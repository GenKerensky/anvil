# Snap layouts

Snap commands detach the active window from tiling and place it at a useful fraction of the current monitor's work area. They are convenient for temporary side-by-side arrangements that should not change the tiling tree.

| Placement                  | Default shortcut                            |
| -------------------------- | ------------------------------------------- |
| Left third                 | <kbd>Ctrl</kbd>+<kbd>Alt</kbd>+<kbd>D</kbd> |
| Left two-thirds            | <kbd>Ctrl</kbd>+<kbd>Alt</kbd>+<kbd>E</kbd> |
| Right third                | <kbd>Ctrl</kbd>+<kbd>Alt</kbd>+<kbd>G</kbd> |
| Right two-thirds           | <kbd>Ctrl</kbd>+<kbd>Alt</kbd>+<kbd>T</kbd> |
| Center at the current size | <kbd>Ctrl</kbd>+<kbd>Alt</kbd>+<kbd>C</kbd> |

Left and right snaps fill the monitor work area's height and apply Anvil's configured gaps. Centering preserves the window's current width and height.

Press <kbd>Super</kbd>+<kbd>C</kbd> when you want the snapped window to return to its tiled layout.
