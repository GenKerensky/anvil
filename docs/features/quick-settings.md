# Quick Settings

Anvil adds a **Tiling** tile to GNOME's Quick Settings menu.

## Tile controls

- Select the tile to toggle global tiling mode.
- Open the tile's menu to toggle **Gaps Hidden when Single**.
- Toggle **Show Focus Hint Border**.
- Toggle **Move Pointer with the Focus**.
- Select **Settings** to open the full Anvil preferences window.

The settings action is hidden while GNOME's session mode does not allow preferences, such as on the lock screen.

## Show or hide the tile

Use **Anvil Preferences → Appearance → Anvil in quick settings**. The tile is shown by default; its visibility and global tiling state are stored independently.
